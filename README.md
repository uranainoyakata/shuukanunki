# しょうじの週間運気予報

「ミニマリスト・禅」をテーマにした、週間運気予報サイトです。

## 構成
- `index.html`: サイトの骨組み
- `style.css`: デザイン（禅スタイル）
- `script.js`: データ読み込み・表示ロジック
- `data.json`: 週間データ（運勢、吉日、メッセージ）

## 更新方法
毎週の更新は `data.json` を書き換えるだけで完結します。

### 1. 期間の更新
`"period": "2026.02.16 - 02.22"` を書き換えます。

### 2. 吉日の更新
`calendar` 配列内の各要素の `lucky` リストを書き換えます。
- 例: `"lucky": ["一粒万倍日", "大安"]`

### 3. 月別運勢の更新
`fortunes` 配列内の各月の `word` (キーワード), `text` (詳細), `item` (ラッキーアイテム) を書き換えます。

## GitHubへの公開と運用（推奨）
1. **GitHubリポジトリを作成**（例：`weekly-fortune`）
2. **ファイルをアップロード**
3. **GitHub Pagesを有効化**（Settings > Pages で `main` ブランチを選択）
   - これにより、世界中にサイトが公開されます。
   - 以降、`data.json` をGitHub上で編集・コミットするだけで、サイトが自動的に最新情報に更新されます。
